1. Basic Import Types 

# Direct import
import os
# Usage: os.path.join()

# From import
from os import path
# Usage: path.join()

# Star import
from os import *
# Usage: join() (if join was exported)>
2. Howfrom fastcore.all import *Works

When you use a star import; python will...

  1. Find the module
  2. Executes the module runs the code in all.py
  3. Imports the symbols: Brings in everything from __all__ (or will use public names)
