Python Import Mechanics

September 16, 2025

Basic Import Types

  • Direct Import import os

    Usage: os.path.join()

  • From Import from os import path

    Usage: path.join()

  • Star Import from os import *

    Usage: join()

    Note: will import all methods or methods in the __all__ when that exists

Barrel Export

aka a "re-export module"

Some packages have a module that is soley for importing other modules

from fastcore.all import * 

        function hello() {
            console.log("Hello, world!");
        }
