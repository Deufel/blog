Technical Details

Key Features:

  • Algorithmic Generation: 16 colors from 3 variables (hue, chroma, rotation)
  • Gray Mixing: Subtle color mixing with grays for sophisticated surfaces
  • Automatic Dark Mode: Uses light-dark() for seamless theme switching
  • Progressive Enhancement: More color in interactive elements, less in backgrounds
  • Accessibility First: Maintains contrast ratios while adding personality
  • Easy Theming: Change one hue value to completely rebrand
